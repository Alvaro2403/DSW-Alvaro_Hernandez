Contenido:

CSS: El CSS es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado. Es muy usado para establecer el diseño visual de los documentos web, e interfaces de usuario escritas en HTML o XHTML

propiedades básicas de CSS:

Estas son las propiedades principales de CSS:

  • Font-family: Define la familia tipográfica y es el tipo de letra o familia de tipos de letra es el tipo de letra que se aplicará al cuerpo de texto en un navegador.
    El tipo de letra es relevante para la visualización del texto en la pantalla, en una impresora o en otro tipo de dispositivos.

  • Font-size: Es la que define el tamaño que tendrá la fuente y este valor podrá escribirse en pixels o en ems.

  • Color: Define el color que llevara la tipografía, para usar los colores se pueden referir de 3 formas diferentes, siendo con sistema Hexadecimal ejemplo: #FF0000, con los nombres de los colores, esta opción es más limitada y los nombres de los colores deben ponerse en inglés, por ejemplo: black, pink, Green. O usando el sistema de RGB el cual te permite hacer transparencias agregando un canal alfa.

  • Max-width o mind-width: esta etiqueta define el ancho máximo o mínimo que tendrá un elemento.

  • Height: Define el alto de un elemento, y se puede escribir en forma de pixeles, ems o porcentaje.

  • Max height o min height: define el alto máximo o mínimo de un elemento.

  • Padding: Esta etiqueta define la distancia desde el borde de un elemento hasta su contenido.

  • Margin: define la distancia entre un elemento y otro.

  • Border: define el border que va a tener un elemento, su color, su estilo y grosor

  • Backgound: Define el fondo de los objetos; este puede ser una imagen o un color solido o degradado.
