Inline:

Elementos en línea no empiezan en una nueva línea y solo ocuparan el ancho que sea necesario. El elemento span es un ejemplo de los elementos en línea.

Elemento SPAN:

1. El elemento span es un contenedor en línea que se utiliza para marcar una parte de un texto o una parte de un documento.

2. El elemento span tiene atributos no son necesarios, pero style, class y id son comunes.
